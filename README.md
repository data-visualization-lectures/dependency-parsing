# 日本語係り受け解析可視化ツール

ウェブブラウザ上で日本語の文章を解析し、係り受け関係を視覚的に表示するツールです。

## 機能

- **形態素解析**: kuromoji.js による日本語形態素解析
- **係り受け推定**: ルールベースの係り受け関係推定
- **視覚化**: SVG による美しい係り受け図の表示
- **インタラクティブ**: ホバーやクリックで詳細情報を表示
- **エクスポート**: SVG/PNG 形式でダウンロード可能

## 使い方

1. ブラウザで `index.html` を開く
2. テキストエリアに日本語の文章を入力
3. 「解析する」ボタンをクリック
4. 係り受け図が表示されます

### ビルド

npm run build

docsをサーブしながらブラウザで確認
```
npm run serve:docs
```

またはビルドから一連で実行
```
npm run build:serve
```

### ローカルサーバーで実行

```bash
npx http-server docs -p 3000 -o

```


## 技術スタック

- **HTML5**: セマンティックマークアップ
- **CSS3**: モダンなダークテーマデザイン
- **JavaScript**: Vanilla JS (フレームワーク不要)
- **kuromoji.js**: 日本語形態素解析ライブラリ (CDN経由)

## 注意事項

- このツールは簡易的なルールベースの係り受け推定を使用しています
- より高精度な解析には CaboCha や GiNZA などの専門ツールをご利用ください
- ブラウザで直接開く場合、CORS の制限により kuromoji の辞書読み込みに失敗する可能性があります。その場合はローカルサーバーを使用してください
